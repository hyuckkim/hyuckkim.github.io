[Github](https://github.com/hyuckkim/ten2two)

C++ 닷넷 윈폼.

위의 문자열과 진수를 설정해서 출력할 수 있다.  
입력값이 이상하다면 출력 창에 오류 메시지가 표시된다.

'문자 수정...' 버튼을 누르면
10보다 높은 진수에서 사용할 문자 세트를 사용자 지정할 수 있는 윈도우가 열린다.  
문자 세트는 export / import 할 수 있다.

## 사족
C++이랑 윈폼이 흔히 같이 쓰이는 조합은 아닌데
(숙련자들은 win32로 직접 쓰고 아니면 C#)
이건 내가 visual studio 템플릿을 뒤지다가 뭣도 모르고 골라서 그렇다.  
MDSN을 보고
어, 나는 `Form::`으로 쓰고 있는데 여기서는 `Form.`으로 쓰네 하고
당황했던 적이 있다.  
당황이 아니라 틀렸다고 생각했었다.

나름 정성들여서 UI 늘리는 거 막고 아이콘도 그려보고 재밌게 만들었다.

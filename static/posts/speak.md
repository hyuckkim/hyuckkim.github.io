C++ GDI+로 만든 투명 이미지 뷰어.

## 기능
두 개의 exe 파일이 제공된다.

### speak.exe
png 또는 gif 이미지를 이 파일에 끌어 놓으면 실행할 수 있다.  
이미지는 오른쪽 아래에 창 없이, 항상 맨 앞에 표시된다.

왼쪽 클릭 + 드래그로 이동,  
오른쪽 클릭으로 종료,  
휠 스크롤로 확대 / 축소,  
휠 클릭으로 대칭 전환 기능이 있다.

이미지를 파일에 끌어 놓는 건 cli 상에서는 2번째 인수 (`[1]`)에 넣는 걸 말한다.  
다른 인수도 있다.

`--pos x y`: 오른쪽 아래 대신 지정한 좌표 (좌상단 기준) 에 표시한다.  
`--width w` : 기본 너비를 w로 지정한다. 높이는 너비와 이미지 비율에 따라 조절됨.  
`--height h` : 기본 높이를 h로 지정한다. 너비는 높이와 이미지 비율에 따라 조절됨.  
width와 height를 둘 다 사용하면 height가 무시된다.  
`--xreversed`, `--yreversed` : 각각 x, y축으로 대칭해서 이미지를 표시한다.

이러면 배치 파일에서 원하는 이미지를 원하는 위치에 표시할 수 있다.
```powershell
./speak.exe .\스피키.gif --pos 50 500 --height 300 --xreversed
```
이런 식으로.

### embed.exe
이미지를 speak.exe에 '삽입' 해서 '그냥' 실행할 수 있게 만든다.  
이미지를 embed.exe에 끌어 넣으면 speak.exe를 인식해서,
이미지 데이터와 각종 헤더 정보를 뒤에 덧붙인다.

## 사족
[love2d](https://www.love2d.org)에서,
게임 실행할 때 폴더를 실행 파일에 끌어서 실행하는게 정말 인상깊었다.  
빌드하는 것도 그냥 명령어로 파일을 합치는 식으로 하고.

원래는 이미지를 '뒤집어서' 넣었었는데  
(이러면 원래 파일은 exe로 쓰고 뒤집은 파일은 온전한 이미지가 된다.
이러면 헤더가 필요없다)  
파일 이름같은 각종 정보를 저장하려고 하다보니까 굳이 뒤집어야 할 이유가 없어져서
그냥 쓰기로 했다.
